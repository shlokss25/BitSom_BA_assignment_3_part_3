# Python Assignment Part 3

This assignment is about working with file handling, APIs and exception handling in Python.

## What I did

In this project, I created a Python script that:

* Writes and reads data from a text file
* Fetches product data from an API (DummyJSON)
* Filters and sorts products based on rating and price
* Handles different types of errors using try-except
* Logs errors into a separate file

## Files in this project

* part3_api_files.py → main Python script
* python_notes.txt → created by the program (contains notes)
* error_log.txt → stores error logs with timestamp
* README.md → this file

## How to run

1. Install requests library (if not installed):

   pip install requests

2. Run the Python file:

   python part3_api_files.py

3. Follow the inputs shown in the terminal

## Notes

* The API used is: https://dummyjson.com
* The POST request is just simulated (data is not actually saved)
* Error logs are automatically added to error_log.txt
